# Roadmap

## Short-term
- Add schema viewer in the Streamlit sidebar.
- Cache schema introspection with auto-refresh button.
- Improve prompt with multilingual support (Hindi, Telugu).

## Medium-term
- Add translation pipeline for non-English queries.
- Pagination for large result sets in the frontend.
- Expanded validation (component-level checks, PICARD-style decoding).

## Long-term
- Role-based access control for different users.
- Support safe, controlled data modifications (insert/update) with audit logs.
- Scale to larger schemas and benchmark datasets (Spider, WikiSQL, MultiSpider).
